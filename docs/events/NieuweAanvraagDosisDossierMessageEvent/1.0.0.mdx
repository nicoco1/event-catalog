## Overview
Het `NieuweAanvraagDosisDossierMessageEvent` wordt verzonden wanneer er een nieuwe aanvraag is geregistreerd en er een dosis dossier moet worden aangemaakt. 
Dit event volgt na `NieuweAanvraagNotificatieMessageEvent`
