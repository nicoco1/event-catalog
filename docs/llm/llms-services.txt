# Services

## [AanvragenService - 1.0.0](https://eventcatalog.dev/docs/services/AanvragenService/1.0.0.mdx)

Service voor het beheren van aanvragen

### Sends
- [Nieuwe Aanvraag Geregistreerd - 1.0.0](https://eventcatalog.dev/docs/events/NieuweAanvraagGeregistreerdEvent/1.0.0.mdx) - Wordt getriggerd wanneer een nieuwe aanvraag (zonder bestemming) is geregistreerd in de aanvragenservice.
- [Bestemming Geregistreerd - 1.0.0](https://eventcatalog.dev/docs/events/BestemmingVoorAanvraagGeregistreerdEvent/1.0.0.mdx) - wordt getriggerd wanneer een bestemming is geregistreerd voor een aanvraag.
- [Nieuwe Aanvraag Notificatie Initiated - 1.0.0](https://eventcatalog.dev/docs/events/NieuweAanvraagNotificatieMessageEvent/1.0.0.mdx) - Event dat wordt verzonden om een notificatie te versturen bij een nieuwe aanvraag. Dit is de ontvangstbevestiging voor de aanvrager (NieuweAanvraagNotificatieMessageEvent).
- [Nieuwe Aanvraag Dosis Dossier Initiated - 1.0.0](https://eventcatalog.dev/docs/events/NieuweAanvraagDosisDossierMessageEvent/1.0.0.mdx) - Event dat wordt verzonden om een dosis dossier aan te maken bij een nieuwe aanvraag (NieuweAanvraagDosisDossierMessageEvent).
- [Dosis Dossier Created - 1.0.0](https://eventcatalog.dev/docs/events/NieuweAanvraagDosisInfoMessageEvent/1.0.0.mdx) - Event dat wordt verzonden om dosis info zoals dosis dossier referte op de aanvraag bij te houden (NieuweAanvraagDosisInfoMessageEvent).

### Receives
- [Registreer Aanvraag Kotlabel - 1.0.0](https://eventcatalog.dev/docs/events/RegistreerAanvraagKotlabelApiRequest/1.0.0.mdx) - Maakt een nieuwe kotlabel aanvraag aan in de aanvragenservice.
- [Nieuwe Aanvraag Geregistreerd - 1.0.0](https://eventcatalog.dev/docs/events/NieuweAanvraagGeregistreerdEvent/1.0.0.mdx) - Wordt getriggerd wanneer een nieuwe aanvraag (zonder bestemming) is geregistreerd in de aanvragenservice.
- [Bestemming Geregistreerd - 1.0.0](https://eventcatalog.dev/docs/events/BestemmingVoorAanvraagGeregistreerdEvent/1.0.0.mdx) - wordt getriggerd wanneer een bestemming is geregistreerd voor een aanvraag.

### Writes to
- Does not write to any containers

### Reads from
- Does not read from any containers

## [IntegrationProcess FormulierenLoket - 1.0.0](https://eventcatalog.dev/docs/services/IntegrationProcessFormulierenLoket/1.0.0.mdx)

Service voor de integratie met het FormulierenLoket om aanvragen te verwerken

### Sends
- [Registreer Aanvraag Kotlabel - 1.0.0](https://eventcatalog.dev/docs/events/RegistreerAanvraagKotlabelApiRequest/1.0.0.mdx) - Maakt een nieuwe kotlabel aanvraag aan in de aanvragenservice.
- [Upload Document Request - 1.0.0](https://eventcatalog.dev/docs/events/UploadDocumentRequest/1.0.0.mdx) - Upload aanvraag bijlage van het FormulierenLoket naar VLOK als document.

### Receives
- [Formulier Aanvraag Kotlabel Submitted - 1.0.0](https://eventcatalog.dev/docs/events/FormulierAanvraagKotlabelSubmittedEvent/1.0.0.mdx) - Wordt getriggerd wanneer een gebruiker een kotlabel aanvraag indient via het FormulierenLoket.
- [Bestemming Geregistreerd - 1.0.0](https://eventcatalog.dev/docs/events/BestemmingVoorAanvraagGeregistreerdEvent/1.0.0.mdx) - wordt getriggerd wanneer een bestemming is geregistreerd voor een aanvraag.

### Writes to
- Does not write to any containers

### Reads from
- Does not read from any containers
