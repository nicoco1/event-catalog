## Overview

De `RegistreerAanvraagKotlabelApiRequest` command wordt gebruikt om een nieuw kotlabel aanvraagformulier te verwerken dat is ingediend via het FormulierenLoket. 
Wanneer dit command wordt ontvangen, zal de AanvragenService de formuliergegevens ophalen en een nieuwe aanvraag aanmaken in het systeem.
